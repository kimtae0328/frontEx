함수

1) 선언적 함수

function 키워드에 함수명을 작성하여 선언 하는 방법

2) 익명의 함수

함수명을 작성하지 않고 변수에 대입하는 방법

3) 스스로 실행하는 함수

함수 정의와 동시에 즉시 실행되는 함수로 한번만 호출되며 다시호출 할 수 없다

4) 화살표 함수

        ES6에 도입된 함수 선언 방법으로 
        function 키워드 대신 => 를 사용해 좀 더 
        간략한 방법으로 함수를 선언할 수 있다.

        [표현법]
            - 매개 변수가 없을 때 : () => { ... }
            - 매개 변수가 하나 일 때 : x => { ... }
            - 매개 변수가 두 개 이상일 때 : (x, y) => { ... }
            - 리턴 처리
                - 처리할 라인이 여러 개라면 {}로 감싸서 처리해야 한다.
                (x, y) => {
                    실행 구문;

                    return x * y;
                }
                - 처리할 라인이 하나라면 {}와 return 문을 생략할 수 있다.
                    코드블럭과 리턴문을 생략 하여 작성 가능
                (x, y) => x * y;

2. 함수의 매개변수

1) 매개변수

함수 실행에 필요한 값을 외부에서 함수로 전달할 때 매개 변수를 통해서 매개값을 전달한다.
함수를 호출할 때 매개 변수의 개수만큼 전달하는 것이 일반적이지만 그렇지 않은 경우에도 에러가 발생하지 않는다.
